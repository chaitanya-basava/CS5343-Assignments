# CS5343 Project 3
This repo contains the solutions for project 3 of CS5343 Algorithm Analysis and Data Structures

## Execute project
Run the following commands to execute
1. unzip sxb220302.zip
2. cd ./sxb220302
3. javac -sourcepath ./ -d ./classes ./MDS.java ./P3Driver.java
4. java -cp classes sxb220302.P3Driver {path to testcases text file}
