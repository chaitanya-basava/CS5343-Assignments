# CS5343 Project 1
This repo contains the solutions for project 1 of CS5343 Algorithm Analysis and Data Structures

## Execute assignment
Run the following commands to execute each assignment
1. unzip sxb220302.zip
2. cd ./sxb220302
3. javac -sourcepath ./ -d ./classes/ ./Expression.java
4. java -cp classes sxb220302.Expression {path to testcases text file}
