# CS5343 Project 2
This repo contains the solutions for project 2 of CS5343 Algorithm Analysis and Data Structures

## Execute project
Run the following commands to execute
1. unzip sxb220302.zip
2. cd ./sxb220302
3. javac -sourcepath ./ -d ./classes/ ./Timer.java ./AVLTree.java ./AVLTreeDriver.java ./BinarySearchTree.java
4. java -cp classes sxb220302.BinarySearchTree {path to testcases text file}
5. java -cp classes sxb220302.AVLTreeDriver {path to testcases text file}
